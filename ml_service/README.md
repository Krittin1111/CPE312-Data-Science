CPE312 Docker Runner

โปรเจกต์นี้เป็นตัวอย่างการใช้งาน Docker สำหรับรันแอปพลิเคชันในรายวิชา CPE312 โดยมีทั้งส่วนของเว็บแอปและสคริปต์ที่เกี่ยวกับการประมวลผลหรือฝึกโมเดล

โครงสร้างโปรเจกต์
- Dockerfile — ไฟล์ที่ใช้สำหรับ build Docker image  
- app.py — โค้ดของแอปพลิเคชันหลัก  
- train_model.py — สคริปต์สำหรับฝึกโมเดล (ถ้ามีการใช้งาน)  
- requirements.txt — รายชื่อไลบรารีที่ต้องติดตั้ง  

การ Build Image
ให้เปิด terminal แล้วเข้าไปยังโฟลเดอร์โปรเจกต์ จากนั้นใช้คำสั่ง:

```
docker build -t cpe312-docker-runner .
```

การรัน Container
```
docker run -p 5000:5000 cpe312-docker-runner
```

จากนั้นเข้าใช้งานได้ที่:

http://localhost:5000

หมายเหตุเพิ่มเติม
- ตรวจสอบว่าไฟล์ requirements.txt มีรายการไลบรารีครบ  
- หากต้องการปรับการฝึกโมเดล ให้แก้โค้ดหรือ Dockerfile ตามต้องการ
